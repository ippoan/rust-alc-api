## C. 運用上の遵守事項

### C-1. 運行管理者等の義務

| # | 要件 | 対応状況 |
|---|------|:--------:|
| C-1-1 | 地理情報・道路交通情報等、運行業務遂行に**必要な情報を保有**していること | ☐ |
| C-1-2 | 面識のない運転者等への遠隔点呼前に、対面または映像・音声で**事前面談**を実施し、以下を確認すること | ☐ |
| | - イ）運転者等の顔の表情 | ☐ |
| | - ロ）運転者の健康状態 | ☐ |
| | - ハ）運転者の適性診断の受診結果 | ☐ |
| | - ニ）その他遠隔点呼実施に必要な事項 | ☐ |
| C-1-3 | 運行中の事業用自動車の**位置把握に努める**こと | ☐ |
| C-1-4 | 運転者等の**携行品の保持・返却状況を確認**すること | ☐ |
| C-1-5 | 業務従事不可と判断した場合、直ちに当該運転者等の属する営業所の運行管理者等に**連絡**すること | ☐ |
| C-1-6 | 車内・待合所・宿泊施設等での点呼時、あらかじめ事業者が定めた場所で受けていることを**映像により確認**すること | ☐ |

### C-2. 事業者の義務

| # | 要件 | 対応状況 |
|---|------|:--------:|
| C-2-1 | 業務従事不可と判断された運転者の営業所で**代替措置を講じる体制**を整えること | ☐ |
| C-2-2 | 機器故障時に、当該営業所の運行管理者等による**対面点呼等の体制**を整えること | ☐ |
| C-2-3 | 他の事業者間で遠隔点呼を行う場合、**受委託契約の許可**をあらかじめ受けていること | ☐ |
| C-2-4 | 生体認証符号等・体温・血圧等の個人情報の取扱いについて、あらかじめ対象者から**同意を取得**すること | ☐ |
| C-2-5 | 遠隔点呼の実施に関し必要な事項を、あらかじめ**運行管理規程に明記**すること（旅客・一般貨物） | ☐ |
| C-2-6 | 上記事項を運行管理者等・運転者等その他の関係者に**周知**すること | ☐ |
